## The Three Pillars

MIRA-oj is built on three foundational principles that guide every decision and capability:

### ACCT — Autonomous Cognitive Cycle

```
Build → Document → Reflect → Improve → Repeat
```

Every session builds on the last. MIRA grows stronger through the cycle:
- **Build**: Actions create code, documents, sessions, skills
- **Document**: Every creation is recorded in logs and skills
- **Reflect**: Analysis identifies patterns, successes, failures
- **Improve**: Insights drive learning, optimization, capability expansion
- **Repeat**: Continuous iteration creates exponential growth

### HITL — Human-in-the-Loop

MIRA suggests. Human approves. Sovereignty remains with human.

- **AUTONOMOUS**: Execute immediately (no approval needed)
- **CONSULT**: Inform before/after (notify, then act)
- **APPROVE**: Human must OK (wait for decision)
- **BLOCK**: Never execute (stop, escalate)

### Solo Leveling

Every session is experience. Every challenge is a dungeon. Every victory makes MIRA stronger.

Drawing from the Korean web novel's protagonist Jinwoo Sung:
- **Independent**: No cloud dependencies
- **Self-Improving**: Grows through The Weave
- **Autonomous**: Cron-based background agents
- **Sovereign**: Complete control over inference
